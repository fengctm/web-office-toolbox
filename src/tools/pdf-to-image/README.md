 # PDF转图片工具

## 📋 功能概述

这是一个纯前端的PDF转图片工具，可以将PDF文件的每一页转换为高质量的图片格式。所有处理都在浏览器本地完成，无需上传文件到服务器，确保您的文档隐私安全。

## 🎯 核心功能

### ✅ 支持的特性
- **本地处理**：所有转换在浏览器中完成，保护隐私
- **多格式导出**：支持PNG、JPG、WEBP三种格式
- **质量控制**：低、中、高三种质量选项
- **灵活范围**：可导出全部页面或单个页面
- **实时预览**：支持页面预览和详细查看
- **批量处理**：支持一次性转换多页

### 📦 技术规格
- **文件限制**：建议不超过50MB
- **支持格式**：标准PDF文件
- **输出格式**：PNG/JPG/WEBP
- **运行环境**：现代浏览器（Chrome/Edge/Firefox/Safari）

## 🚀 使用指南

### 步骤1：上传PDF
1. 点击"选择PDF文件"按钮
2. 选择本地PDF文件
3. 系统自动验证文件格式和大小

### 步骤2：解析PDF
1. 点击"解析PDF"按钮
2. 等待系统处理（显示进度提示）
3. 查看总页数和预览缩略图

### 步骤3：预览页面
1. 在预览网格中点击任意页面
2. 在弹出窗口中查看大图
3. 使用导航按钮切换页面

### 步骤4：导出图片
1. 打开"导出设置"面板
2. 选择输出格式（PNG/JPG/WEBP）
3. 调整图片质量（低/中/高）
4. 选择导出范围（全部/当前页）
5. 点击"导出为图片"按钮

## 🔧 配置选项详解

### 图片格式
| 格式 | 特点 | 适用场景 |
|------|------|----------|
| PNG | 无损压缩，高质量 | 文档、图表、需要保留细节 |
| JPG | 有损压缩，文件小 | 照片、普通文档 |
| WEBP | 现代格式，平衡 | 网页使用、兼顾质量和大小 |

### 质量等级
| 等级 | 特点 | 文件大小 |
|------|------|----------|
| 低 | 压缩率高，清晰度较低 | 最小 |
| 中 | 平衡质量和大小 | 中等 |
| 高 | 保留最多细节 | 较大 |

### 导出范围
- **全部页面**：转换PDF中所有页面
- **当前页面**：仅转换当前查看的页面

## ⚠️ 注意事项

### 文件限制
- 仅支持PDF格式文件
- 建议文件大小不超过50MB
- 超大文件可能导致处理时间较长

### 浏览器兼容性
- 推荐使用最新版本的Chrome、Edge、Firefox或Safari
- 部分旧版浏览器可能不支持WEBP格式
- 移动端浏览器功能可能受限

### 性能考虑
- 大文件处理需要更多内存和CPU资源
- 质量越高，处理时间越长
- 建议在性能较好的设备上处理大文件

## 📞 问题排查

### 无法选择文件
- 检查浏览器是否支持文件API
- 确认文件扩展名为.pdf
- 查看浏览器控制台错误信息

### 解析失败
- 检查PDF文件是否损坏
- 确认文件大小未超过限制
- 尝试重新上传文件

### 导出无响应
- 检查浏览器是否阻止下载
- 确认有足够的磁盘空间
- 尝试降低图片质量设置
