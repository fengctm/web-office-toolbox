# 图片转 PDF 工具

## 📋 功能概述

这是一个纯前端的图片转 PDF 工具，可以将多张图片合并为一个 PDF 文件。支持图片排序、预览和多种页面尺寸设置。所有处理都在浏览器本地完成，无需上传文件到服务器。

## 🎯 核心功能

### ✅ 支持的特性

- **本地处理**：所有转换在浏览器中完成，保护隐私
- **多格式支持**：支持 JPG、PNG、WEBP 等图片格式
- **拖拽上传**：支持拖拽和点击选择多张图片
- **图片排序**：支持手动排序和按名称自动排序
- **实时预览**：支持图片预览和页面导航
- **多种尺寸**：支持 A4、原始尺寸、适应屏幕三种页面尺寸
- **批量处理**：支持一次性转换多张图片

### 📦 技术规格

- **图片限制**：单张建议不超过50MB
- **支持格式**：JPG、PNG、WEBP、GIF 等标准图片格式
- **输出格式**：PDF
- **运行环境**：现代浏览器（Chrome/Edge/Firefox/Safari）

## 🚀 使用指南

### 步骤1：上传图片

1. 点击"选择图片"按钮或拖拽图片到上传区域
2. 支持多选，可一次性选择多张图片
3. 系统自动验证图片格式和大小

### 步骤2：管理图片

1. 在"图片列表"中查看已添加的图片
2. 使用方向按钮调整图片顺序
3. 点击页码可跳转到指定位置
4. 可按文件名自动排序
5. 可删除不需要的图片

### 步骤3：预览图片

1. 切换到"预览"标签页
2. 左侧显示缩略图导航
3. 右侧显示完整预览
4. 点击缩略图可快速跳转

### 步骤4：导出 PDF

1. 切换到"导出设置"标签页
2. 设置文件名（默认：图片转PDF）
3. 选择页面尺寸：
    - A4：标准A4纸张尺寸 (210×297mm)
    - 图片原始尺寸：保持图片原始比例
    - 适应屏幕：自动调整到合适大小
4. 点击"导出 PDF"按钮

## ⚠️ 注意事项

### 文件限制

- 支持所有标准图片格式
- 建议单张图片不超过50MB
- 图片数量无严格限制，但建议控制在50张以内

### 浏览器兼容性

- 推荐使用最新版本的Chrome、Edge、Firefox或Safari
- 部分旧版浏览器可能不支持WEBP格式
- 移动端浏览器功能可能受限

### 性能考虑

- 大图片处理需要更多内存和CPU资源
- 图片越多，处理时间越长
- 建议在性能较好的设备上处理大量图片

## 🛠️ 技术实现

### 使用的库

- **pdf-lib**：PDF文档创建和图片嵌入
- **file-saver**：文件下载
- **Vuetify**：UI组件库

### 核心流程

1. **图片读取**：使用FileReader读取图片为DataURL
2. **图片加载**：创建Image对象获取尺寸信息
3. **PDF创建**：使用pdf-lib创建空白PDF文档
4. **页面创建**：根据配置创建对应尺寸的页面
5. **图片嵌入**：将图片嵌入到PDF页面
6. **文件下载**：生成PDF文件并触发下载

## 📞 问题排查

### 无法选择文件

- 检查浏览器是否支持文件API
- 确认文件格式为图片
- 查看浏览器控制台错误信息

### 导出失败

- 检查图片是否损坏
- 确认文件大小未超过限制
- 尝试减少图片数量

### 下载无响应

- 检查浏览器是否阻止下载
- 确认有足够的磁盘空间
- 尝试使用其他浏览器

## 🔒 隐私与安全

### 本地处理保证

- ✅ 文件不会上传到任何服务器
- ✅ 处理过程中不会发送网络请求
- ✅ 所有数据在浏览器内存中处理
- ✅ 页面刷新后无残留数据

### 数据清理

- 组件卸载时自动清理临时数据
- 不会在本地存储中留下文件痕迹
- 可随时清除浏览器缓存确保隐私
