# 图片转图标工具

## 功能说明

将普通图片（JPEG、PNG、WEBP）裁切为正方形，然后生成各种尺寸和格式的图标文件。

## 主要特性

### 1. 图片裁切
- 支持 1:1 正方形裁切
- 支持旋转、翻转操作
- 支持拖拽上传
- 实时预览裁切效果

### 2. 多种格式
- **PNG**：通用格式，支持透明背景
- **WebP**：现代格式，体积更小，质量更好
- **ICO**：Windows 网站图标格式（包含多尺寸）
- **SVG**：矢量格式（封装 PNG）

### 3. 丰富的尺寸预设
- **标准尺寸**：16, 32, 48, 64, 128, 256
- **扩展尺寸**：512, 1024
- **iOS**：60, 76, 120, 152, 167, 180
- **Android**：48, 72, 96, 144, 192, 512

### 4. 高级设置
- 圆角处理：0-50% 可调
- 图片质量：60-100% 可调（仅 WebP）
- 背景颜色：透明/白色/彩色

### 5. 灵活导出
- 单个图标下载
- 按格式打包下载
- 一键 ZIP 打包全部
- 自动生成 README 文件
- 生成 Favicon HTML 代码

## 技术实现

### 核心依赖
- **cropperjs** - 图片裁切
- **png-to-ico** - ICO 格式生成
- **jszip** - ZIP 打包
- **file-saver** - 文件下载

## 使用流程

1. **上传图片**：点击上传区域或拖拽图片
2. **裁切调整**：调整裁切区域，确保主体居中
3. **选择尺寸**：使用快速预设或自定义尺寸
4. **选择格式**：选择需要生成的图标格式
5. **调整设置**：设置圆角、质量等参数
6. **生成图标**：点击生成按钮，等待处理完成
7. **预览下载**：预览生成的图标并下载

## 注意事项

- 支持的图片格式：JPEG、PNG、WebP
- 图片大小限制：最大 10MB
- SVG 格式说明：封装 PNG，非真正矢量图
- ICO 格式说明：自动包含多个尺寸
- 推荐圆角范围：10-25%
