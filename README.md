# Web Office Toolbox - 纯前端工具箱

## 📋 项目需求与定位

### 核心目标
创建一个**纯前端的、模块化的、可扩展的** Web 工具箱平台，无需后端依赖，所有功能在浏览器中本地运行。

### 🎯 核心需求

#### 1. 纯前端架构
- ✅ **零后端依赖**：完全基于浏览器运行，无需 Node.js 后端服务
- ✅ **本地文件处理**：所有文件操作在浏览器内存中完成
- ✅ **隐私安全**：用户数据不会离开本地浏览器
- ✅ **离线支持**：可部署为静态网站，支持离线访问

#### 2. 模块化工具系统
- ✅ **工具独立**：每个工具是独立的 Vue 组件模块
- ✅ **即插即用**：添加新工具只需配置和创建组件文件
- ✅ **自动注册**：通过配置文件自动发现和注册工具
- ✅ **热插拔**：可以启用/禁用工具，无需重启

#### 3. 优秀用户体验（PC端优先）
- ✅ **Material Design**：基于 Vuetify 3 的现代化界面
- ✅ **流畅动画**：页面切换、卡片悬停、按钮交互都有动画
- ✅ **响应式设计**：完美适配移动端、平板、桌面端
- ✅ **深色模式**：自动适配系统主题，支持手动切换

#### 4. 开发友好
- ✅ **清晰文档**：详细的开发指南和示例代码
- ✅ **快速上手**：3 步即可添加新工具
- ✅ **热重载**：开发时即时预览
- ✅ **JavaScript**：纯 JavaScript 开发，无需 TypeScript

## 🏗️ 系统架构

### 技术栈
```
前端框架：Vue 3 + Composition API
UI 组件库：Vuetify 3 (Material Design)
构建工具：Vite
路由管理：Vue Router 4
开发语言：JavaScript
```

### 核心组件
```
src/
├── config/
│   └── tools-config.js      # 工具配置系统（核心）
├── pages/
│   ├── Home.vue             # 首页（工具展示网格）
│   └── ToolPage.vue         # 工具详情页容器
├── components/
│   ├── ToolCard.vue         # 工具卡片组件
│   ├── ToolGrid.vue         # 工具网格布局
│   └── PageTransition.vue   # 页面过渡动画
├── tools/                   # 工具模块目录
│   ├── README.md            # 工具开发文档
│   └── example-counter/     # 示例工具
├── router.ts                # 路由配置
├── main.js                  # 应用入口
└── App.vue                  # 根组件
```

### 数据流
```
工具配置 → 工具注册器 → 首页展示 → 用户点击 → 路由跳转 → 工具渲染
```

## 🎨 设计理念

### 1. 极简主义
- **配置即代码**：一个配置文件管理所有工具
- **约定优于配置**：标准化的目录结构和命名
- **零配置启动**：安装依赖后立即可用

### 2. 模块化设计
- **工具解耦**：工具之间完全独立，无相互依赖
- **单一职责**：每个工具只做一件事
- **接口清晰**：统一的组件接口和配置规范

### 3. PC端优先
- **桌面优化**：大屏幕布局优先设计
- **大卡片展示**：舒适的卡片大小和间距
- **鼠标友好**：悬停效果和点击区域优化
- **响应式扩展**：移动端作为扩展支持

## 🛠️ 添加新工具（3步）

### 步骤 1：添加配置
在 `src/config/tools-config.js` 中添加：
```javascript
{
  code: 'my-tool',
  name: '我的工具',
  icon: 'mdi-toolbox',
  description: '工具描述',
  enabled: true,
  component: () => import('../tools/my-tool/index.vue')
}
```

### 步骤 2：创建目录
```bash
mkdir src/tools/my-tool
touch src/tools/my-tool/index.vue
```

### 步骤 3：实现组件
```vue
<template>
  <v-container fluid>
    <h2>我的工具</h2>
    <!-- 工具内容 -->
  </v-container>
</template>

<script setup>
// 工具逻辑
</script>
```

**完成！** 工具会自动出现在首页。
